# Delivery Readme

这是一个高校辅导员备考辅助 Skill为解决知识点琐碎没有体系的问题不再需要自己反复找资料、辨别资料、整理题目、设计复习顺序，而是可以直接进入学习、练习和纠错，
里面包含政策学习、可用题库训练、案例分析、结构化面试、公文写作和答案批改等内容几乎涵盖所有知识点以及考点。拿到后建议先看 README.md 和 quick-start.md，直接复制里面的指令开始用。
题库来源包括培训资料、模拟题、真题回忆。
- 压缩包名称：1.4.2-usable-question-bank.zip
- 压缩包内容：`college-counselor-prep/` Skill 目录，包含 SKILL.md、README、quick-start、source-disclaimer、usable 题库、索引、模板、评分规则、工作流和报告。
- 用途：面向中国高校辅导员招聘考试备考者的通用备考辅助 Skill。
- 适合交付给：正在备考高校辅导员招聘考试、需要题库训练和答题批改的同学。
- 交付前应提醒：题库不是官方真题库，部分题目需要人工核对，具体考试以目标高校公告和官方政策为准。
- 不应承诺：不承诺押题、包过或替代官方公告。
 拿到后如何开始：先阅读 README.md 和 quick-start.md，再按 quick-start 中的指令开始训练。
- 制定 30 天或 14 天备考计划
- 从可用题库中出题训练
- 批改简答题、论述题、案例分析和面试回答
- 讲解辅导员岗位、学生工作和思政教育相关考点
- 组织每日训练和错题复盘
- 辅助公文写作训练
- 复习 2025-2026 时政材料

这个Skill可以充当三个角色：
资料管理员
从几千条内容中找到相关题目和知识，不需要手动翻文件
出题老师
按题型、考点、难度和学校来源进行训练或组卷。
陪练与批改老师
简答题、案例分析和结构化面试回答进行评价，并继续针对薄弱点训练。
这个Skill可以形成：
学习知识 → 抽题练习 → 回答 → 评分 → 找出薄弱点 → 再出针对性题目 → 模拟考试
